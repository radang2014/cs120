Query to Get All Songs from a Genre:

The genre I selected was: Pop

The query I used was: {genre:"Pop"}

Part 2: Shell

Selecting the Database: (the database name is ps3-3)

Command: use ps3-3

Showing All Collections in the Database:

Command: show collections

Showing All Data for All Documents

Command: db.songs.find()

Showing Title, Artist, and Genres Corresponding to Each Song

Command: db.songs.find({}, {_id: 0, title: 1, artist: 1, genre: 1})

Showing Titles for Particular Genre

The genre I selected was: Pop

Command: db.songs.find({genre: "Pop"}, {_id: 0, title: 1})

Showing Title and Artist for Songs Belonging to Two Specific Genres

The two genres I selected were: Film and Soundtrack

Command: db.songs.find({genre: {$all: ["Film", "Soundtrack"]}}, {_id: 0, title: 1, artist: 1})
