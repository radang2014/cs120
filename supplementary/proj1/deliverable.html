    • This page is about a fictional band but attempts to feel realistic.

    • Not counting media queries, there are 30 CSS rules in an external style sheet, which can be found here.
    • The homepage has 2 internal CSS rules.
    • The about page has 2 internal CSS rules.
    • The upcoming events page has 3 internal CSS rules.
    • The store page has 3 internal CSS rules.
    • This totals to 40 CSS rules when not counting media queries.
  • A form
    • A form is present on the store page, designed for ordering merchandise and/or tickets.
  • A table
    • A table is present on the upcoming events page, showing information about upcoming events.
    • A "hidden" table is also present in the order form on the store page.
  • A list
    • A list is present on the homepage containing the name and role of each band member.
    • There are also multiple lists on the credits page listing out acknowledgements and/or sources.
  • An external stylesheet
    • External stylesheet can be found here. All pages import this stylesheet.
  • An internal stylesheet on at least one page
    • Internal stylesheets can be found on all pages except for the contact page and the credits page.
  • Two images with at least one background image
    • Background Images:
      • American Flag Stars (on every page)
      • Group Photo of Band Members (on Home, About Us, and Contact)
    • Not Background Images:
      • Merch T-Shirt (on Home and Store)
      • Merch Beanie (on Home and Store)
      • Photo of "Emily" (on About Us)
      • Photo of "Mike" (on About Us)
      • Photo of "Ben" (on About Us)
      • Photo of "Chris" (on About Us)
    • All image sources are on the credits page.
  • Two fixed page elements (on every page)
    • Banner at top of page with link to home page
    • Navigation bar with links to each page (except for credits)
    • Footer with link to home page and credits
